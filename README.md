# mac_setup
My configuration files to setup mac

## installation & usage

<pre>
git clone https://github.com/kenteroshima/mac_setup.git
mv mac_setup/macsetup.sh ~/
sudo sh macsetup.sh
</pre>

## file locations

|file name |location |
|----------|----------|
|Rprofile |~/.Rprofile |
|vimrc |~/.vimrc |
|zshrc |~/.zshrc |
